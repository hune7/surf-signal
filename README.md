# 발리 레기안 해변 3일 서핑 추천 시간 예측기

발리 레기안 해변 라인업 기준 좌표인 위도 `-8.711`, 경도 `115.166`과 송정해수욕장 라스트웨이브 앞바다 좌표인 위도 `35.1791`, 경도 `129.2000`을 사용해 **오늘, 내일, 모레**의 시간별 바람, 파도, 조수 데이터를 가져옵니다.

웹사이트 상단에서 **발리 레기안**과 **송정 라스트웨이브**를 선택할 수 있습니다. 프로그램은 각 시간대마다 서핑하기 좋은지 나쁜지 평가하고, 파도 높이를 기준으로 **초보 / 중급 / 고수** 난이도도 함께 출력합니다.

## 1. API 키 안내

이 프로젝트는 **Open-Meteo 무료 API**를 사용합니다.

- API 키 발급이 필요 없습니다.
- 회원가입 없이 바로 실행할 수 있습니다.
- 해양 데이터 출처: [Open-Meteo Marine API](https://open-meteo.com/en/docs/marine-weather-api)
- 바람 데이터 출처: [Open-Meteo Forecast API](https://open-meteo.com/en/docs)

## 2. 설치 방법

PowerShell에서 프로젝트 폴더로 이동한 뒤 아래 명령어를 실행하세요.

```powershell
py -m pip install -r requirements.txt
```

## 3. 실행 방법

### 웹사이트로 보기

로컬에서 내려받아 사용할 때는 아래 파일을 더블클릭하면 됩니다.

```text
open_website.bat
```

또는 아래 파일을 브라우저로 직접 열어도 됩니다.

```text
web/index.html
```

브라우저에서 열면 발리 현재 시간이 자동으로 표시되고, 오늘부터 모레까지 3일치 서핑 예보를 바로 불러옵니다.

### 콘솔로 보기

```powershell
py main.py
```

## 4. 평가 기준

- 오프쇼어 바람: 풍향각 45도 이상 135도 이하
- 강풍 제외: 풍속 25km/h 초과 제외
- 파도 주기: 7초 이상
- 미드 타이드: 날짜별 최저 수위와 최고 수위 사이의 중간 수위 전후
- 초보 난이도: 파도 높이 1m 이하
- 중급 난이도: 파도 높이 1m 초과 2m 미만
- 고수 난이도: 파도 높이 2m 이상급

단순히 파도 높이만 보지 않고, 바람 방향과 풍속, 파도 주기, 조수 조건을 함께 조합해 `좋음`, `애매`, `나쁨`으로 평가합니다.

## 5. 스팟별 바람 기준

- 발리 레기안: 동풍 계열 45도 이상 135도 이하, 풍속 25km/h 이하
- 송정 라스트웨이브: 서풍~북풍 계열 270도 이상 360도 이하, 풍속 7m/s 이하에 해당하는 25.2km/h 이하
